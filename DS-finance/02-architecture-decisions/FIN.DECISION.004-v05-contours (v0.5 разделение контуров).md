# FIN.DECISION.004 — v0.5 разделение контуров

## Решение

`DS-finance` разделяется на два финансовых контура:

- `business-finance` (финансы бизнеса);
- `personal-finance` (личные финансы).

## Принцип

Контуры не смешиваются.

Взаимодействие между ними проходит только через явные трансферы:

- `transfer_out` (вывод денег из бизнеса);
- `transfer_in` (ввод денег в бизнес).

## Ограничения

- запрещено учитывать личные расходы в бизнесе;
- запрещено учитывать бизнес-активы в личных расходах;
- все движения между системами фиксируются явно.

## Примеры классификации

- растения для сада -> `personal.expense` (личный расход);
- ремонт дома -> `personal.project` (личный проект);
- пристройка -> `personal.asset` (личный актив после завершения).
