# DS-finance — Домен финансов

## Назначение

Отдельный домен для финансового различения системы создания по всем активным
и будущим контурам экосистемы.

Это не бухгалтерия и не один локальный `Pack`. Это домен, который удерживает:

- `cash discipline`
- `finance views`
- `project funding`
- `financial agents`

## Главный предмет домена

Не "финансы вообще", а финансы системы создания:

- что усиливает систему;
- что только отнимает ресурс;
- где узкое место;
- где инвестиция создаёт будущую мощность;
- где трата не возвращает ни деньги, ни capability.

## Структура домена

| Путь | Назначение |
|------|------------|
| `context.md` | Текущий контекст домена и принятые архитектурные решения |
| `01-concepts/` | Базовые финансовые различения домена |
| `02-architecture-decisions/` | Мелкие карточки решений по архитектуре домена |
| `PACK-finance-views/` | Финансовые представления по доменам |
| `PACK-cash-discipline/` | Правила расходов и темпа трат |
| `PACK-project-funding/` | Финансирование проектов и gap analysis |
| `PACK-finance-agents/` | Роли и контракты финансовых агентов |

## Source of truth

- первичное знание остаётся в родных доменах;
- `DS-finance` хранит их финансовые представления и verdict-ы;
- weekly financial constraints materialize здесь и в `DS-strategy`.
