## Правило W17

### Weekly portfolio rule

Для этой недели принимается рабочее правило:

- `1 домен = 1 главный активный РП`
- остальные задачи домена = очередь домена, а не параллельный активный фокус
- неделя управляется как портфель доменов, а не как плоский список из всех хвостов сразу

Это правило пока считается не полностью автоматизированным capability, а управленческим решением Стратега на W17.

## Отдельная тема W17 — Парк / труба

Цель недели не «думать о трубе», а перевести вопрос из неофициального и тревожного статуса в рабочий управляемый контур.

Минимальный результат недели:
- собран единый пакет следов по трубе;
- понятен следующий официальный источник verdict;
- заказана экспертиза или подготовлен формальный запрос;
- есть decision tree по вариантам `ничья / чужая / перенос / только отступ`.
