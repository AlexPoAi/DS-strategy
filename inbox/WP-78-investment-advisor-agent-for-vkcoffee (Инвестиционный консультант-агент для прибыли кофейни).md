# Контекст

Нужен отдельный агент-консультант по инвестициям, который знает проект, стратегию владельца и помогает принимать решения по распределению прибыли кофейни:
- куда выгодно инвестировать внутри проекта,
- где создавать дополнительный заработок,
- что оптимизировать в текущей модели.

# Цель

Создать агента уровня `Investment Advisor`, который даёт регулярные, проверяемые финансово-управленческие рекомендации по капиталу и росту.

# Контракт агента (v1)

1. Анализирует:
   - прибыль/маржу по направлениям,
   - окупаемость инициатив,
   - риски и ликвидность,
   - альтернативы вложений (внутри кофейни и рядом).
2. Формирует:
   - еженедельный shortlist инвест-решений,
   - сценарии `base / optimistic / conservative`,
   - рекомендации по доп. доходу.
3. Контролирует:
   - факт выполнения инвест-решений,
   - план-факт по ожидаемому эффекту.

# Scope

1. Описать профиль агента и память контекста владельца (цели, стиль риска, горизонты).
2. Определить входные данные (продажи, затраты, склад, маркетинг, кэш).
3. Сделать шаблон invest-card:
   - гипотеза,
   - бюджет,
   - ожидаемый эффект,
   - срок окупаемости,
   - риски,
   - решение (`go/hold/stop`).
4. Запустить пилот на 3-5 реальных инвест-гипотезах.

# Acceptance

1. Агент создан в агентстве и имеет чёткий контракт.
2. Есть минимум 3 готовые invest-card по реальным инициативам.
3. Рекомендации выражены цифрами и сроками, а не общими советами.
4. Есть weekly-ритм пересмотра портфеля решений.
