# WP-50: Каталог готовых блюд из кассы Сабы

## Описание

Создать структурированную карточку номенклатуры готовых блюд кухни на основе выгрузки из Сабы (Каталог_2026-03-29.xlsx). Каталог станет базовым артефактом для цепочки: ТТК → себестоимость → сроки хранения → регламент маркировки.

## Артефакт

`KITCHEN.WP.011 (Каталог готовых блюд из кассы Сабы).md` в `PACK-kitchen/04-work-products/`

## Контекст

- Источник: `knowledge-base/Отчёты для бота/Каталог_2026-03-29.xlsx` (выгрузка Сабы)
- Связан с: KITCHEN.METHOD.007 (сроки хранения), KITCHEN.METHOD.002 (списание)
- Следующий РП по цепочке: дополнение ТТК и себестоимости по каждой позиции
- Производство: Пн / Ср / Пт, выдача к 18:00

## Структура артефакта

Для каждой позиции из раздела «Еда» кассы Сабы:
- Название (точно как в кассе)
- Категория (онигурадзу / ролл / блины / боул / горячее / салат / прочее)
- Ключевые ингредиенты (лимитирующие для срока)
- Срок хранения (из KITCHEN.METHOD.007)
- Ссылка на ТТК (если есть)
- Себестоимость (если есть в калькуляциях)

## Критерий готовности

- [ ] Все позиции раздела «Еда» из Сабы внесены
- [ ] Напротив каждой — срок хранения
- [ ] Отмечено: есть ТТК или нет (пробел = следующий РП)
- [ ] Файл в PACK-kitchen/04-work-products/
