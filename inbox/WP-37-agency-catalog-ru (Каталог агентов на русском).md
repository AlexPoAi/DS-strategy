# РП#37: Каталог агентов на русском языке

## Цель

Создать русский каталог всех агентов из agency-agents с описанием для каких задач они подходят. Каталог будет использоваться для быстрого поиска нужного специалиста.

## Нанятый агент

**Technical Writer** (если есть) или работаем без агента

## Что нужно сделать

### 1. Извлечь метаданные всех агентов

Из каждого `*.md` файла извлечь:
- `name` — название агента
- `description` — описание
- `emoji` — эмодзи
- `vibe` — краткая суть

### 2. Перевести на русский

Создать структуру:
```markdown
## 🏗️ Backend Architect (Архитектор бэкенда)

**Для чего:** Проектирование масштабируемых систем, архитектура баз данных, API, облачная инфраструктура

**Когда нанимать:**
- Нужно спроектировать микросервисную архитектуру
- Оптимизация производительности БД
- Разработка API с версионированием
- Построение event-driven систем

**Файл:** `engineering/engineering-backend-architect.md`
```

### 3. Организовать по категориям

Категории:
- **Инженерия** (engineering)
- **Дизайн** (design)
- **Маркетинг** (marketing)
- **Продукт** (product)
- **Продажи** (sales)
- **Стратегия** (strategy)
- **Специализированные** (specialized)
- **Тестирование** (testing)
- **Управление проектами** (project-management)

### 4. Создать поисковый индекс

JSON-файл для быстрого поиска:
```json
{
  "agents": [
    {
      "name": "Backend Architect",
      "name_ru": "Архитектор бэкенда",
      "category": "engineering",
      "keywords": ["api", "база данных", "микросервисы", "масштабирование"],
      "file": "engineering/engineering-backend-architect.md"
    }
  ]
}
```

## Структура файлов

```

```

## Границы

### Входит
- Извлечение метаданных из всех агентов
- Перевод описаний на русский
- Организация по категориям
- JSON-индекс для поиска

### Не входит
- Полный перевод содержимого агентов
- Интеграция с Telegram-ботом (это РП-36)

## Критерий завершения

- Каталог на русском со всеми 75+ агентами
- Каждый агент: название, описание, когда нанимать, файл
- JSON-индекс для программного поиска
- Организовано по категориям
