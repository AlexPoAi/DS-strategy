# WP-121 — Личный DS саморазвития и контур энергии

## Зачем открыт

Во время стратегической сессии выявлен новый критичный личный контур:

- нет устойчивого слота саморазвития;
- есть неудовлетворённость этим провалом;
- есть проблемы по питанию;
- мало энергии;
- снижается продуктивность.

Это уже не "личная заметка на потом", а управленческий фактор,
который влияет на качество weekly execution по всей экосистеме.

## Цель slice

Materialize новый `DS`-контур, посвящённый:

- саморазвитию;
- саморефлексии;
- личному состоянию;
- энергии;
- питанию;
- личным переживаниям и внутренним сигналам.

## Что должно materialize

1. Каноническое имя нового личного `DS`.
2. Его назначение и границы:
   - что туда складывается;
   - что туда не относится.
3. Первый bounded weekly use-case.
4. Понимание, как этот контур отражается в Obsidian-мозге.
5. Решение, какой один главный ход недели делаем по этому контуру.

## Truthful scope

В этот slice входит:

- определение нового личного `DS`;
- фиксация его роли в weekly-портфеле;
- запуск Obsidian-папки / human-layer зоны;
- выделение первого сигнального focus-area: энергия / питание / слот саморазвития.

В этот slice не входит:

- полная перестройка личности через один заход;
- весь backlog саморефлексии;
- комплексная health-system без отдельного плана.

## Предварительный weekly use-case

Первый честный use-case для этого контура:

`Понять, почему проседают энергия и продуктивность, и вернуть в неделю хотя бы один устойчивый слот саморазвития.`

Выходом должны стать не общие мысли, а:

- один главный дефицит;
- один weekly-ритуал;
- одна Obsidian-зона для накопления личных сигналов;
- одно ограничение, которое нельзя игнорировать при планировании недели.

## Acceptance

1. Новый `DS` назван и описан.
2. Для него создан human-layer дом в Obsidian.
3. Определён первый weekly focus.
4. Личный контур встроен в weekly portfolio, а не оставлен вне системы.

## Следующий честный шаг

После открытия `WP-121`:

1. решить каноническое имя контура;
2. создать его Obsidian-дом;
3. зафиксировать первый weekly вопрос:
   - энергия;
   - питание;
   - слот саморазвития;
4. не расширять этот контур дальше, пока не пройден первый bounded cycle.
