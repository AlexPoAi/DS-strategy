# Контекст

Нужно провести целевой анализ проекта `VK-offee` и определить, какие автоматизации внедрять в первую очередь, чтобы получить быстрый измеримый эффект по прибыли, качеству управления и снижению ручной нагрузки.

# Цель

Собрать приоритизированную карту автоматизаций с оценкой `влияние / трудозатраты / риск / окупаемость`.

# Обязательные домены анализа

1. **Продажи**: отчёты, воронка, повторные покупки, средний чек, товарные аномалии.
2. **Маркетинг**: офферы, сегменты, кампании, локальные спец-предложения, LTV/retention.
3. **Менеджмент**: контроль задач, план-факт, статусы контуров, отчётность владельцу.

# Что ещё обязательно включить

4. **Склад и закупки**: прогноз спроса, reorder, ABC/маржа, анти-перезатарка.
5. **Финансы**: cashflow по точкам, P&L light, алерты по убыточным позициям.
6. **HR/операции смен**: графики, чек-листы, дисциплина выполнения регламентов.
7. **Клиентский сервис**: SLA ответа, качество сервиса, обратная связь и повторные визиты.
8. **Интеграции/боты**: Telegram/RAG/Google Drive маршруты и их надежность.

# Scope

1. Инвентаризация текущих процессов: где уже есть автоматизация, где ручной труд.
2. Карта болей: потери денег/времени/качества.
3. Shortlist автоматизаций (10-20 инициатив) с ранжированием.
4. План внедрения в 3 волны:
   - Wave 1: быстрые победы (1-2 недели),
   - Wave 2: структурные улучшения (2-6 недель),
   - Wave 3: стратегические контуры (6+ недель).

# Acceptance

1. Готов документ с таблицей автоматизаций и приоритетами.
2. По каждой инициативе есть owner, артефакт, metric и deadline.
3. Минимум 3 автоматизации из Wave 1 готовы к запуску без доп. уточнений.

# Next slice (today)

1. Собрать текущие контуры и дыры по 8 доменам.
2. Выдать shortlist top-10 по ROI.
3. Зафиксировать план Wave 1 для старта в текущей неделе.
