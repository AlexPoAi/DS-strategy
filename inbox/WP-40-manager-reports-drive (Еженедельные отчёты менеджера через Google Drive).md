# WP-40: Еженедельные отчёты менеджера через Google Drive

## Суть задачи

Менеджер (Жанна) еженедельно снимает остатки и работает в Saby Presto.
Задача: выстроить простой канал передачи отчётов из Saby → Google Drive →
стратегическая сессия с анализом через Claude.

**Принцип:** никакого кода и API — только папка на Drive, договорённость о формате и триггер в протоколе стратегирования.

---

## Схема работы

```
Жанна (Saby Presto)
    │ раз в неделю выгружает отчёты
    ↓
📁 Google Drive: VK-offee/Еженедельные отчёты/W{N} YYYY-MM-DD/
    │   ├── накладные-реестр.xlsx
    │   ├── выручка-по-точкам.xlsx
    │   ├── остатки-на-точках.xlsx
    │   └── (опционально) расходы-хозяйка.xlsx
    ↓
Claude читает папку на сессии стратегирования
    │ триггер: «читай отчёты недели» / «проверяй Drive»
    ↓
Анализ: флаги аномалий, выручка по точкам, топ/антитоп, остатки
```

---

## Что нужно выгружать из Saby (список для Жанны)

### Обязательно (еженедельно)

| Отчёт | Что содержит | Формат |
|-------|-------------|--------|
| **Реестр накладных** | Все поступления за неделю: поставщик, сумма, дата, точка | xlsx/csv |
| **Выручка по точкам** | Самокиша / Тургенева / Луговая — по дням и итог за неделю | xlsx/csv |
| **Остатки на точках** | Текущие остатки по позициям на каждой точке | xlsx/csv |

### Желательно (раз в месяц или по запросу)

| Отчёт | Что содержит |
|-------|-------------|
| **Расходы (хозяйка/хорика)** | Одноразовая посуда, химия, расходники |
| **Зарплата / ФОТ** | Начисления сотрудникам за период |
| **Списания** | Что списали, на какую сумму, по каким причинам |
| **ТОП продаж** | Рейтинг позиций по количеству и выручке |
| **Себестоимость** | Маржа по ключевым позициям (если Saby считает) |

### Для полного анализа кофейни (расширенный контур)

| Показатель | Откуда | Периодичность |
|-----------|--------|--------------|
| Средний чек по точке | Saby | Еженедельно |
| Количество транзакций | Saby | Еженедельно |
| Конверсия допродаж | Saby (если есть) | Еженедельно |
| Рейтинг сотрудников по выручке | Saby | Еженедельно |
| Инвентаризация (остатки) | Saby / вручную | Раз в 2 недели |
| Штрафы / возвраты / скидки | Saby | Еженедельно |
| Поставщики: исполнение по накладным | Saby / вручную | Еженедельно |
| Загрузка по часам (пик) | Saby | Раз в месяц |

## Шаги реализации

### Шаг 1: Жанна создаёт папку (5 мин)
1. Открыть Google Drive
2. Создать папку: `VK-offee / Еженедельные отчёты`
3. Внутри создать подпапку по формату: `W13 2026-03-29`
4. Открыть доступ для Александра (редактор или только просмотр)
5. Прислать ссылку на папку `Еженедельные отчёты`

### Шаг 2: Настроить sync (15 мин)
- Добавить путь к папке в `sync_google_drive.py` или рядом
- Проверить что файлы из этой папки попадают в `~/Github/VK-offee/weekly-reports/`

### Шаг 3: Добавить триггер в протокол стратегирования (15 мин)
- В `memory/protocol-open.md` или в сессионный протокол добавить:
  ```
  Триггер «читай отчёты недели»:
  → Открыть ~/Github/VK-offee/weekly-reports/
  → Найти папку последней недели
  → Проанализировать по чеклисту (WP-40)
  ```

### Шаг 4: Первый прогон (30 мин)
- Жанна выгружает первый пакет отчётов
- Проверяем что формат читается
- Проводим первый анализ вместе

## Чеклист анализа на сессии стратегирования

```
□ Выручка по точкам: сравнить с прошлой неделей, флаги ±20%
□ Топ-5 позиций по выручке (совпадает с ожиданием?)
□ Антитоп-5 позиций (кандидаты на вывод из меню?)
□ Накладные: расхождения суммы / поставщик-нарушитель?
□ Остатки: критически низкие / критически высокие позиции
□ Средний чек: динамика по точкам
□ Аномалии: нехарактерные дни (провалы / пики) → почему?
```

## Сообщение для Жанны

> Жанна, привет! Хотим наладить простую систему: раз в неделю (воскресенье или понедельник утром) ты выгружаешь из Сабы три отчёта и кладёшь их в папку на Google Drive. Мы потом читаем и анализируем на стратегической сессии.
>
> **Что нужно:**
> 1. Создай папку на Google Drive: `VK-offee / Еженедельные отчёты`
> 2. Каждую неделю создавай подпапку: `W14 2026-04-05` (номер недели + дата понедельника)
> 3. Туда клади три файла из Сабы:
>    - **Реестр накладных** за неделю (все поступления)
>    - **Выручка по точкам** (Самокиша / Тургенева / Луговая)
>    - **Остатки на точках**
> 4. Открой доступ к папке `Еженедельные отчёты` для [email Александра]
> 5. Пришли ссылку на папку
>
> Формат файлов — как Саби выгружает (xlsx или csv), ничего конвертировать не нужно. Спасибо!
