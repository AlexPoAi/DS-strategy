# WP-47: Система учёта токенов с Telegram-отчётами

## Цель

Создать автоматическую систему учёта токенов Claude Code:
1. Сбор токенов из всех активных сессий (2 окна)
2. Расчёт стоимости в долларах
3. Автоматическая отправка отчёта в Telegram в конце дня

## Проблема

- Утечка токенов ($900 за 5 дней, 6289 запросов)
- Нет мониторинга расхода в реальном времени
- Невозможно отследить аномалии

## Решение

### Компоненты

1. **collect-tokens.sh** — сбор токенов из `~/.claude/projects/`
2. **calculate-cost.py** — расчёт стоимости по тарифам Anthropic
3. **send-token-report.sh** — отправка отчёта в Telegram
4. Интеграция в `close-task.sh` — автозапуск при закрытии дня

### Тарифы Anthropic (март 2026)

| Модель | Input | Output | Cache Write | Cache Read |
|--------|-------|--------|-------------|------------|
| Sonnet 4.6 | $3/MTok | $15/MTok | $3.75/MTok | $0.30/MTok |
| Opus 4.6 | $15/MTok | $75/MTok | $18.75/MTok | $1.50/MTok |
| Haiku 4.5 | $0.80/MTok | $4/MTok | $1.00/MTok | $0.08/MTok |
