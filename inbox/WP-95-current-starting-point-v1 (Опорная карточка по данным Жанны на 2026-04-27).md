## Что из этого уже понятно

1. Полная работа по роли операционного менеджера шире, чем складской учет.
2. Внутри роли смешаны разные слои: склад, касса, отчеты, найм, процессы, поручения.
3. Жанна начала с правильного для себя формата: визуальной карты.
4. Для передачи человеку одной карты недостаточно: нужны инструкции, чеклисты и критерии готовности.
5. Наш Phase 1 должен остаться узким: склад, накладные, инвентаризация, Честный знак, остатки, отчетность.
6. Авито, регламенты, широкий Telegram и поручения нельзя автоматически включать в стартовую роль нового человека.

## Что уже можно считать подтвержденным

Подтверждено:

- у роли есть повторяемые циклы;
- складской учет действительно является ядром;
- инвентаризация и накладные являются регулярными процессами;
- Честный знак относится к операционной зоне;
- отчетность состоит из конкретных файлов и выгрузок;
- часть Saby/кассы уже вынесена в скринкасты;
- Жанна понимает необходимость записать и транскрибировать скринкасты.

## Чего не хватает

Критические недостающие элементы:

1. Инструкция по инвентаризации.
2. Инструкция по накладным: приход, списание, перемещение, электронные накладные.
3. Инструкция по Честному знаку.
4. Правило работы с недостачами и боем.
5. Чеклист отчетности: какие файлы, откуда брать, куда класть, кто проверяет.
6. Правила работы с остатками по складу для заказа.
7. Список доступов и инструментов.
8. Критерии готовности по каждой регулярной операции.
9. Разделение текущей полной работы по роли и стартовой роли нового человека.

## Управленческий вывод

Сейчас у нас уже достаточно данных, чтобы не говорить “мы не знаем, что делает операционный менеджер”.

Но данных еще недостаточно, чтобы безопасно заменить человека без участия Жанны.

Статус на сейчас:

- карта роли: есть;
- видео по части процессов: есть;
- текстовая система роли: не готова;
- инвентаризационный блок: критически не заполнен;
- границы стартовой роли нового человека: нужно держать уже, чем полная карта.

> Мы наблюдаем, как Жанна показывает фактическую работу по роли операционного менеджера, фиксируем предоставленные материалы, отделяем карту работы от инструкций и превращаем подтвержденные зоны в проверяемую систему для нового человека.

1. Не расширять роль нового человека на всю карту.

3. Подготовить короткий запрос Жанне на подтверждение и дозаполнение: инвентаризация, накладные, Честный знак, отчетность, остатки.
