# WP-44: Система работы с отчётами Saby

## Цель

Спроектировать систему автоматической обработки отчётов из Saby Presto для RAG-бота:
1. Какие отчёты нужны
2. Как их структурировать
3. Как индексировать для поиска
4. Какие сценарии использования

## Контекст

Менеджер Жанна раз в 2 недели (после инвентаризации) скидывает:
- Накладные от поставщиков (все за период)
- Отчёты по продажам
- Каталог продукции (с остатками)

Бот должен отвечать на вопросы:
- "Найди накладную по Хорике"
- "Цена одноразовых стаканов"
- "Менялась ли цена на стаканы?"
- "Сколько продали капучино в марте?"

## Часть 3: Сценарии использования бота

### Сценарий 1: Поиск накладной по поставщику

**Запрос:** "Найди накладную по Хорике"

**Что делает бот:**
1. Ищет в ChromaDB по ключевым словам: "Хорика", "накладная"
2. Находит карточки INV-* с supplier="Хорика"
3. Возвращает список накладных с датами и суммами

**Ответ:**
```
Найдено 3 накладных от Хорики:

1. 25.03.2026 — 15,420 руб (12 позиций)
2. 11.03.2026 — 8,750 руб (8 позиций)
3. 04.03.2026 — 12,300 руб (10 позиций)

Что показать подробнее?
```

---

### Сценарий 2: Проверка цены товара

**Запрос:** "Цена одноразовых стаканов"

**Что делает бот:**
1. Ищет в каталоге продукции: "стакан одноразовый"
2. Находит актуальную цену из последнего каталога
3. Показывает цену закупки

**Ответ:**
```
Стакан 350 мл (одноразовый):
- Цена закупки: 8.50 руб/шт
- Упаковка: 50 шт
- Последнее обновление: 30.03.2026
```

---

### Сценарий 3: История изменения цен

**Запрос:** "Менялась ли цена на стаканы?"

**Что делает бот:**
1. Ищет все накладные со "стакан"
2. Сравнивает цены по датам
3. Показывает динамику

**Ответ:**
```
История цен на стакан 350 мл:

04.03.2026 — 8.20 руб
11.03.2026 — 8.20 руб (без изменений)
25.03.2026 — 8.50 руб (+3.7%)

Цена выросла на 30 коп за 3 недели.
```

---

### Сценарий 4: Анализ продаж

**Запрос:** "Сколько продали капучино в марте?"

**Что делает бот:**
1. Ищет в отчётах по продажам: "капучино", "март"
2. Суммирует данные за период
3. Показывает статистику

**Ответ:**
```
Капучино 0.3 в марте 2026:

Продано: 2,480 шт
Выручка: 297,600 руб
Средняя цена: 120 руб

Динамика:
- 1-15 марта: 1,240 шт
- 16-30 марта: 1,240 шт (стабильно)
```

## Часть 4: План реализации

### Этап 1: Инфраструктура (1h)
1. Создать папки на VPS
2. Настроить права доступа
3. Создать структуру в VK-offee репо

### Этап 2: Парсер отчётов (2h)
1. Скрипт для парсинга PDF накладных
2. Скрипт для парсинга Excel (продажи, каталог)
3. Генерация карточек MD

### Этап 3: Индексация (1h)
1. Обновить indexer.py для новых карточек
2. Переиндексировать ChromaDB
3. Тестирование поиска

### Этап 4: Автоматизация (1h)
1. Webhook или cron для автообработки
2. Уведомления в Telegram при новых отчётах
3. Мониторинг ошибок

**Итого:** ~5h работы
