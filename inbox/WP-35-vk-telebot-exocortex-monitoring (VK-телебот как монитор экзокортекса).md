# РП#35: VK-телебот как монитор экзокортекса

## Цель

Настроить VK-телебот (токен хранится только в runtime `.env`, не в git) как канал мониторинга экзокортекса: уведомления от strategist/scheduler/extractor, статус агентов, напоминания о РП.

## Что нужно сделать

### 1. Создать отдельный чат для мониторинга
- Создать новый чат/группу для экзокортекса
- Добавить VK-телебота в этот чат
- Получить новый `CHAT_ID`
- Обновить `~/.config/aist/env`

### 2. Настроить уведомления от агентов
Агенты должны отправлять уведомления через `notify.sh`:
- **Strategist:** day-plan завершён, week-review завершён
- **Scheduler:** daily-report создан, ошибки агентов
- **Extractor:** inbox обработан, новые captures
- **Synchronizer:** code-scan завершён

### 3. Меню бота
Команды для управления:
- `/status` — статус всех агентов
- `/agents` — список агентов и последний запуск
- `/wp` — активные РП текущей недели
- `/logs` — последние ошибки

### 4. Интеграция с notify.sh
- Проверить, что `notify.sh` корректно читает env
- Добавить вызовы `notify.sh` в strategist/scheduler/extractor
- Протестировать отправку

## Границы

### Входит
- Настройка чата и бота
- Интеграция уведомлений от агентов
- Базовое меню команд

### Не входит
- Q&A по накладным/каталогу (это WP-34)
- Сложная логика обработки команд
- Веб-интерфейс

## Критерий завершения

- VK-телебот добавлен в новый чат мониторинга
- Уведомления от strategist/scheduler/extractor приходят в чат
- Команды `/status`, `/agents`, `/wp` работают
- Токен и chat_id сохранены в `~/.config/aist/env`
