# Поэтапное строительство (Phased Construction)

## 1. Определение

Метод строительства комплекса кофейни в парке через последовательные юридически завершённые этапы. Каждый этап — самостоятельный законченный объект капитального строительства (ОКС).

## 2. Проблема

Полное строительство комплекса (2 этажа + кухня) требует ресурсов, которые невозможно привлечь единовременно. Нужен метод, поз��оляющий начать эксплуатацию объекта до полного завершения строительства.

## 3. IPO-паттерн

| Элемент | Описание |
|---------|----------|
| Входы | Проектная документация (АР, КР), бюджет этапа, разрешение на строительство |
| Обработка | Строительство этапа → ввод в эксплуатацию → юридическое оформление → реконструкция |
| Выходы | Законченный ОКС с техпаспортом, готовый к следующему этапу |

## 4. Этапы

| Этап | Содержание | Юридический статус |
|------|------------|-------------------|
| **Этап 1** | Фундамент под весь комплекс + несущая система + 1 этаж | Новое строительство → ввод в эксплуатацию |
| **Этап 2** | Пристройка кухни | Реконструкция |
| **Этап 3** | Надстройка 2 этажа | Реконструкция |

## 5. Конструктивные требования к Этапу 1

- Отдельные инженерные вводы
- Завершённая кровля
- Пожарные нормы как для законченного объекта
- Отдельный техпаспорт на 1 этаж
- Конструктив предусматривает надстройку без изменения пятна

## 6. Ключевая формулировка для архитектора

> «Нам нужно спроектировать первый этап так, чтобы он был самостоятельным законченным объектом, но конструктивно предусматривал развитие.»
