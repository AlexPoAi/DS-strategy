# WP-102: Парк — официальные запросы по канализационной трубе v1

## Почему открываем

По трубе в Парке уже есть:
- кадастровый след;
- устный контур про `не на балансе у Воды Крыма`;
- проектная митигация `отступ от трубы`.

Но нет двух вещей, без которых нельзя уверенно принимать следующее решение:
- официального письма от профильного технического адресата;
- нормального стартового комплекта документов для внешнего обращения.

## Цель

Материализовать первый исполнительный пакет по трубе:
1. определить первичных адресатов;
2. подготовить готовые черновики обращений;
3. зафиксировать, какие вопросы задаём официально;
4. оставить следующий ход не в голове, а в проектном контуре.

## Что сделано

1. Подтверждён приоритетный маршрут:
   - первый адресат: `ГУП РК "Вода Крыма"`;
   - второй адресат: `Администрация Бахчисарайского района` на районном уровне, а не сельсовет.
2. В `PACK-park-development` создан новый исполнительный work product по этому циклу.
3. Подготовлены два готовых черновика обращений:
   - в `Воду Крыма`;
   - в `Администрацию Бахчисарайского района`.
4. Вопрос по сельсовету truthfully понижен до вторичного/необязательного канала.

## Что считаем результатом этого slice

- Есть готовый текст первого обращения в `Воду Крыма`.
- Есть готовый текст второго обращения в районную администрацию.
- В проекте Парка появилось одно место, откуда можно продолжать этот цикл без реконструкции контекста.

## Следующий практический шаг

1. Дозаполнить шапки писем:
   - ФИО заявителя;
   - почтовый адрес;
   - телефон;
   - email.
2. Приложить к письму:
   - схему/кадастровый фрагмент;
   - ссылку на участок `90:01:070101:767`;
   - при наличии схему проекта с отступом.
3. Отправить сначала запрос в `Воду Крыма`.
4. В тот же цикл отправить запрос в `Администрацию Бахчисарайского района`.
5. После отправки открыть уже следующий bounded slice:
   - tracking ответов;
   - поиск специалиста на выезд;
   - decision tree по итогам ответов.
