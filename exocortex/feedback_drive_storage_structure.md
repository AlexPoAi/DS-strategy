# Google Drive: Структура хранения документов

## Принцип

**Двухуровневое хранение:**
- **Оригиналы** остаются в папках по проектам (Парк, Самокиша, и т.д.)
- **Копии для печати** хранятся в отдельной папке `для печать/` для удобства и быстрого доступа

Это предотвращает потерю информации о том, откуда печатать, и упрощает подготовку документов к подаче.

---

## Структура папки "для печать"

```
📂 для печать/
├── 📂 Парк (WP-73)
│   ├── 📄 PARK.DOC.038-water-crimea-letter-PRINT-COPY
│   └── (другие документы Парка для печати)
├── 📂 Самокиша (WP-16)
│   ├── 📄 WP-16-SIGNAGE-CHECKLIST-PRINT-COPY
│   └── (другие документы Самокиша для печати)
├── 📂 Кухня Тургенева (WP-14)
│   └── (документы для печати)
└── 📂 Другие...
```

---

## Правила

1. **Оригиналы** — всегда остаются в проектных папках (не перемещать)
2. **Копии для печати** — создавать в `для печать/` с суффиксом `-PRINT-COPY`
3. **Актуальность** — если документ обновился, обновить и копию
4. **Ссылки в памятках** — указывать на копию в папке `для печать/` для удобства

---

## Применение

- WP-73 (Парк): Письмо Водоканал → `для печать/Парк (WP-73)/PARK.DOC.038-PRINT-COPY`
- WP-16 (Самокиша): Чеклист вывески → `для печать/Самокиша (WP-16)/WP-16-SIGNAGE-CHECKLIST-PRINT-COPY`
