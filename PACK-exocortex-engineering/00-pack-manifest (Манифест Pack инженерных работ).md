# PACK-exocortex-engineering — Инженерные работы по экосистеме

## Назначение

Паспорт домена «Инженерные работы по экзокортексу».

Здесь хранится история всех ремонтных и инженерных работ по экосистеме:
диагностики, починок, архитектурных решений по стабилизации агентов,
протоколов верификации и накопленных уроков.

> **Почему этот Pack нужен:** После неоднократных инцидентов (агенты без прав +x,
> зависающий day-close, истёкший OAuth, конфликт rebase/hook) стало очевидно:
> ремонтные работы — это домен со своими объектами, методами и типовыми ошибками.
> Без фиксации истории каждый инцидент решается «с нуля».

## Домен

**Система:** Экзокортекс (FMT-exocortex-template + DS-strategy + агенты)
**Граница:** Инженерные работы по поддержанию работоспособности экзокортекса.
Не включает: бизнес-знание VK-offee, стратегирование, парк.
