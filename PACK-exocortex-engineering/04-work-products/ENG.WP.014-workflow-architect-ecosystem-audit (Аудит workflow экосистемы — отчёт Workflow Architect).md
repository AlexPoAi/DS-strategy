## Критические разрывы

### 🔴 WF-3: Pack → RAG-бот (нет автопереиндексации)
- **Проблема:** Обновили VK-offee → ChromaDB на VPS (72.56.4.61) не знает об этом. Нет webhook, нет cron переиндексации.
- **Симптом:** Бот отвечает по устаревшей базе знаний.
- **Решение:** cron или post-receive hook на VPS после push в VK-offee → запуск indexer.py

### 🔴 WF-2: Сессия → INBOX (нет автоматизации)
- **Проблема:** Проведена сессия стратегирования → задачи зафиксированы только в диалоге. Нет агента, который читает итоги и пишет в INBOX.
- **Симптом:** Задачи «утекают» — не сохраняются между сессиями.
- **Решение:** РП 47 (Extractor redesign) — автоматически переносить задачи из сессий в INBOX-TASKS.md

### 🔴 WF-10: Агентство простаивает
- **Проблема:** 75+ агентов в библиотеке, нанято 8. Ни один не вызывается по доменным задачам автоматически.
- **Симптом:** Работа ведётся вручную вместо делегирования специалистам.
- **Решение:** Hiring Protocol — нанять Document Generator + Report Distribution Agent. Встроить в Ритуал согласования.

### 🟡 WF-7: Week-review сломан с 23 марта
- **Проблема:** 429 cost limit при запуске. Модель слишком дорогая для недельного обзора.
- **Решение:** Переключить week-review на Haiku (30 мин работы).
