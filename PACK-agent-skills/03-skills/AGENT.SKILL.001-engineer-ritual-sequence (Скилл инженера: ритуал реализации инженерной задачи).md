# AGENT.SKILL.001 — Скилл инженера

## Назначение

Канонический ритуал выполнения инженерной задачи в экзокортексе без дрейфа.

## Последовательность

1. Диагностика по фактам: логи, health-check, конкретные файлы/entrypoint.
2. Сверка с Цереном: как это реализовано в рабочем upstream-эталоне.
3. Сверка с целевым эталоном нашего контура: как должно быть у нас.
4. Оценка риска перед правками: что может сломаться и как проверим после.
5. Фиксация РП и контекста:
   - `inbox/WP-*.md`,
   - `docs/WP-REGISTRY.md`,
   - `PACK-exocortex-engineering/04-work-products/ENG.WP.*.md`.
6. Реализация правок только после шагов 1-5.
7. Пост-проверка: health-check и целевые проверки сценария.
8. Закрытие: truthful итог, коммит, push, запись в `inbox/INBOX-TASKS.md`.

## Критерий завершения

- Критичная проблема устранена или эскалирована с фактическим блокером.
- Результат подтверждён проверками.
- Хронология сохранена в реестре и инженерных артефактах.
