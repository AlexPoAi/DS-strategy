# AGENT.SKILL.CONCEPT.001 — Контракт скилла агента

## Определение

Скилл агента — это повторяемый, проверяемый сценарий работы по роли,
где зафиксированы шаги, проверки и критерий завершения.

## Минимальная структура скилла

1. Назначение и границы.
2. Каноническая последовательность шагов.
3. Проверки после выполнения.
4. Критерий завершения.
